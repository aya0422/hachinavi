# はちケアNavi

年齢別口腔ケア診断システム

## 概要
はちケアNaviは、年齢に応じた口腔ケア診断を行い、適切なケアグッズを推薦するWebアプリケーションです。

## 機能
- 11区分年齢システム（乳児前期～高齢期）
- 年齢・状態別診断質問
- 口腔内画像表示
- 商品推薦システム
- レスポンシブデザイン

## 技術スタック
- React 18
- TypeScript
- CSS3
- Create React App

## セットアップ
```bash
npm install
npm start
```
